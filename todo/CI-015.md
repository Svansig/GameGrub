## Problem

New global singleton/companion access can be reintroduced without automated guardrails.

## Scope

- In scope:
  - Add a lightweight check that flags new direct singleton/companion usage patterns in changed lines/files.
  - Provide clear remediation guidance in CI output.
- Out of scope:
  - Blocking all existing legacy usage immediately.
