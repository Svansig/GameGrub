## Problem

There is no single boundary for storage operations, so platform reads/writes and policy logic remain scattered.

## Scope

- In scope:
  - Create `app.gamegrub.storage` package with primary `StorageGateway` and focused sub-gateways.
  - Add Android-backed managers and Hilt module wiring.
  - Add provider/entry-point path for static or framework call sites that cannot inject directly.
- Out of scope:
  - Broad caller migration beyond foundation smoke wiring.
