## Problem

Steam path resolution and install-location policy are tightly coupled to static helpers and preferences.

## Scope

- In scope:
  - Move Steam path computations to `StoragePathGateway`.
  - Remove direct path policy and preference reads from Steam-specific callers.
  - Keep runtime behavior parity for install path selection.
- Out of scope:
  - Non-Steam platform path migration.
