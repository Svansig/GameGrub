## Problem

`app.gamegrub.utils` currently mixes true shared helpers with feature/domain owners (auth, container launch, Steam install/runtime, update flows), which obscures ownership and increases cross-layer coupling.

## Scope

- In scope:
  - Complete file-by-file utils ownership classification and decision log.
  - Create migration tickets for every utility that should move to owned boundaries.
  - Track sequencing and dependencies across child tickets.
- Out of scope:
  - Full code migration implementation in one ticket.
