## Problem

`utils/auth` contains workflow owners (OAuth orchestration, key attestation, Play Integrity lifecycle, platform sign-in state aggregation), not generic helpers.

## Scope

- In scope:
  - Move `AuthUrlRedaction`, `KeyAttestationHelper`, `PlatformAuthUtils`, `PlatformOAuthHandlers`, and `PlayIntegrity` to owned auth/security packages.
  - Replace static/singleton call paths with injectable coordinators/gateways where appropriate.
  - Update call sites in app/service/UI layers.
- Out of scope:
  - Changing auth provider API behavior.
