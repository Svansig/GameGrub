## Problem

Shared helper files include mixed ownership: UI-facing platform/source formatting, persona URL builders, and compile-time toast policy enforcement are colocated with generic utilities.

## Scope

- In scope:
  - Split `FormatUtils` into generic format helpers vs UI/source presentation helpers.
  - Split `StringUtils` so persona/profile URL helpers move to Steam/persona ownership.
  - Move `NoToast` policy guard to explicit policy/tooling boundary package.
- Out of scope:
  - Redesigning UI copy/content behavior.
