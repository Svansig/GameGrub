## Implementation Notes (2026-04-09)

External intent/deep-link validation is already in place:

- **IntentLaunchManager.parseLaunchIntent()** validates:
  - Action must be `app.gamegrub.LAUNCH_GAME`
  - app_id must be > 0
  - game_source defaults to STEAM if invalid
  - container_config JSON limited to 50KB max
  - ContainerData fields have regex validation for screenSize, cpuList, videoMemorySize, drives

- **LaunchRequestManager.handleLaunchIntent()**:
  - Ignores intents re-delivered from recents (FLAG_ACTIVITY_LAUNCHED_FROM_HISTORY)
  - Wraps all parsing in try-catch with Timber logging
  - Emits AndroidEvent.ExternalGameLaunch only on valid requests

- **MainActivity.handleLaunchIntent()** delegates to LaunchRequestGateway
