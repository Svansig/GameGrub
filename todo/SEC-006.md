## Implementation Notes (2026-04-09)

Permission audit performed on AndroidManifest.xml:

| Permission | Status | Justification |
|---|---|---|
| ACCESS_NETWORK_STATE | Keep | Required for network connectivity checks |
| ACCESS_WIFI_STATE | Keep | Required for WiFi state queries |
| MODIFY_AUDIO_SETTINGS | Keep | Required for ALSA audio server (ALSAClient.java) |
| RECORD_AUDIO | **Removed** | Unused - no audio recording feature in app |
| MANAGE_EXTERNAL_STORAGE | Keep | Required for custom game scanning on scoped storage |
| FOREGROUND_SERVICE | Keep | Required for Steam/GOG/Epic/Amazon sync services |
| FOREGROUND_SERVICE_DATA_SYNC | Keep | Required for data sync foreground service type |
| INTERNET | Keep | Required for all platform API calls |
| REQUEST_INSTALL_PACKAGES | Keep | Required for runtime APK/install handling |
| POST_NOTIFICATIONS | Keep | Required for download/progress notifications (requested at runtime) |
| VIBRATE | Keep | Required for controller haptic feedback |

Key finding: RECORD_AUDIO was declared but never used anywhere in the codebase. Removed from manifest.
