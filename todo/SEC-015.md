## Problem

External path validation and storage safety checks are inconsistent when implemented across multiple layers.

## Scope

- In scope:
  - Centralize path normalization and validation in storage managers.
  - Add consistent guardrails for traversal, illegal roots, and unsupported volumes.
  - Ensure callers consume validated storage APIs only.
- Out of scope:
  - Broader non-storage input validation.
