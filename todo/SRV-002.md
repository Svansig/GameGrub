# SRV-002 - Move installer/download helpers to SteamInstallDomain

- **ID**: `SRV-002`
- **Area**: `service/steam`
- **Priority**: `P2`
- **Status**: `Backlog`
- **Owner**: `TBD`

## Problem

Installer/download helper methods in SteamService companion should be in SteamInstallDomain:
- `isImageFsInstalled`, `isImageFsInstallable`, `isSteamInstallable`, `isFileInstallable`
- `fetchFile`, `fetchFileWithFallback`
- `downloadImageFs`, `downloadImageFsPatches`, `downloadFile`, `downloadSteam`

## Scope

- In scope:
  - Move imagefs/steam file existence checks to domain
  - Move HTTP fetch utilities to domain or dedicated download utility
  - Move file download async operations to domain
- Out of scope:
  - Any external API changes

## Acceptance Criteria

- [ ] Helper methods removed from SteamService companion
- [ ] Domain exposes equivalent methods
- [ ] Existing callers work unchanged

## Validation

- [ ] Build passes
- [ ] ImageFS download flow works
