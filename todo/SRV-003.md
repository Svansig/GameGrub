# SRV-003 - Remove DepotDownloader import from SteamService

- **ID**: `SRV-003`
- **Area**: `service/steam`
- **Priority**: `P2`
- **Status**: `Backlog`
- **Owner**: `TBD`

## Problem

SteamService imports `DepotDownloader`, `IDownloadListener`, `AppItem`, `DownloadItem` - these are only used for download orchestration which belongs in domain. After SRV-001, these imports should be removable from SteamService.

## Scope

- In scope:
  - Remove DepotDownloader-related imports from SteamService.kt
  - Verify no other usages remain after download extraction
- Out of scope:
  - Import changes in other files

## Acceptance Criteria

- [ ] Import section no longer contains depotdownloader classes
- [ ] Build passes without errors

## Validation

- [ ] `./gradlew compileDebugKotlin` passes
