## Problem

SteamService companion uses service locator pattern (`instance`, `requireInstance`, `getInstanceOrNull`). This should be deprecated in favor of dependency injection. Callers should get SteamService via DI or use domain methods directly.

## Scope

- In scope:
  - Add @Deprecated to instance accessors
  - Track down internal usages
  - Update call sites to use DI or domain methods
- Out of scope:
  - Breaking changes to public API
