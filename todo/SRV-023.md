# SRV-023 - Remove direct SteamService.requireInstance/getInstance usage from pre-launch utilities

- **Status**: `Backlog`

- **Documentation Impact**: `Expected service access pattern documentation updates.`

## Problem

Pre-launch utilities call `SteamService` global instance APIs directly, creating hidden runtime coupling.

## Scope

- In scope:
  - Replace direct service instance calls in target utilities with injected gateways/dependencies.
  - Keep launch behavior equivalent.
- Out of scope:
  - Full container launch architecture rewrite.
