# SRV-026 - Convert mutable global service flags to scoped state holders with explicit ownership

- **Priority**: `P3`
- **Status**: `In Progress`
- **Owner**: `Copilot`
- **Documentation Impact**: `Expected state ownership migration notes updates.`

## Problem

Mutable global flags (`isRunning`, `isStopping`, etc.) are scattered and can be changed from many call sites.

## Scope

- In scope:
  - Introduce scoped state holders with explicit mutation points.
  - Migrate a selected subset of global service flags.
- Out of scope:
  - Complete state-model rewrite.
