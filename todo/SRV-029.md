## Problem

JavaSteam unified services include `FriendMessages`/`FriendMessagesClient`, but GameNative currently does not expose messaging capability. We need to determine whether messaging-adjacent support provides practical value for launch/session workflows.

## Scope

- In scope:
  - Assess whether friend messaging signals can help gameplay/session reliability or troubleshooting.
  - Identify privacy/security implications and Android UX constraints.
  - Estimate implementation complexity and maintenance burden.
  - Produce a `Go / Defer / Drop` decision with rationale.
- Out of scope:
  - Building full chat UI.
