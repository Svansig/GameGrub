# SRV-030 - Evaluate JavaSteam `UserAccount` and `AuthenticationSupport` integration usefulness and feasibility

## Problem

GameNative has active auth flows, but JavaSteam unified services provide additional account/auth support APIs that are not wired. We need to verify whether they reduce auth edge-case failures and simplify maintenance.

## Scope

- In scope:
  - Compare current auth/session behavior with potential `UserAccount`/`AuthenticationSupport` usage.
  - Identify reliability/security wins and migration risks.
  - Estimate implementation effort including adapter/domain boundary changes.
  - Produce a `Go / Defer / Drop` decision with rationale.
- Out of scope:
  - Complete auth subsystem rewrite.
