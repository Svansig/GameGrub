# UI-003 - Remove unmanaged IO scopes from Steam UI paths

- **ID**: `UI-003`
- **Area**: `ui/screen/library/appscreen`
- **Priority**: `P1`
- **Status**: `Done`
- **Owner**: `Sisyphus`

## Problem

`SteamAppScreen` uses unmanaged background scopes for update/download actions. This creates lifecycle and observability risk.

## Scope

- In scope:
  - Route side effects through ViewModel-managed scope.
  - Keep UI file as event dispatcher and state consumer.
- Out of scope:
  - Behavior changes to Steam download semantics.

## Acceptance Criteria

- [x] Target UI file no longer starts unmanaged IO scopes.
- [x] Steam action flow remains functionally equivalent.
- [x] State/effect updates remain consistent with existing UX.

## Progress

- Created `SteamAppScreenViewModel` with lifecycle-aware coroutine handling.
- **Replaced ALL 11 `CoroutineScope(Dispatchers.IO).launch` calls** in `SteamAppScreen` with ViewModel methods:
  - Simple download calls (4 occurrences) → `viewModel.downloadApp`
  - Delete/uninstall call → `viewModel.deleteApp`
  - Language change download → `viewModel.downloadApp`
  - Cloud sync operation → `viewModel.syncUserCloudFiles`
  - DLC download → `viewModel.downloadAppWithDlc`
  - Delete with container cleanup → `viewModel.deleteAppWithContainerCleanup`
  - Verify with conditional cloud sync → `viewModel.verifyAppWithCloudSync`
  - ImageFS installation → `viewModel.installImageFs`
- Added ViewModel methods for complex operations:
  - `downloadAppWithDlc` - Download with specific DLC list
  - `syncUserCloudFiles` - Cloud file synchronization
  - `verifyAppWithCleanup` - Verify/update with marker cleanup
  - `deleteAppWithContainerCleanup` - Delete with container cleanup (enhanced with Result callback)
  - `verifyAppWithCloudSync` - Verify with optional cloud sync (includes container unpacking)
  - `installImageFs` - ImageFS installation with progress callbacks
- **UI-003 is now complete** - all unmanaged IO scopes have been replaced with ViewModel-managed coroutines.
