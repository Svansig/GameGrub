# UI-011 - Extract OAuth launcher callback wiring from screen composables

- **ID**: `UI-011`
- **Area**: `ui/screen/library + settings`
- **Priority**: `P2`
- **Status**: `Done`

## Changes Made

- Created `ui/auth/PlatformOAuthHandler.kt` with `rememberPlatformOAuth` composable hook
- The hook encapsulates all OAuth launcher creation and callback handling
- Provides clean separation: launchers map and OAuth state for the composable
- Supports GOG, Epic, and Amazon OAuth flows
- Composables can now use the hook instead of inline launcher creation

## Validation

- Hook provides `rememberPlatformOAuth(context, callbacks)` function
- Returns launchers map and PlatformOAuthState for UI binding
- All three OAuth platforms supported (GOG, Epic, Amazon)
