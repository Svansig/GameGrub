## Problem

UI paths were performing direct hardware and system reads (`GPUInformation`, battery/thermal/proc/system-service reads) instead of using a shared abstraction.

## Scope

- In scope:
  - Migrate `LibraryViewModel` GPU lookup to injected gateway.
  - Migrate `SteamAppScreen` GPU lookups to gateway provider access.
  - Migrate `PerformanceHudView` runtime telemetry queries to gateway.
- Out of scope:
  - Rewriting unrelated UI lint warnings.
