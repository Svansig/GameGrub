## Problem

Settings flows directly mutate storage preferences and storage-selection state without storage-package mediation.

## Scope

- In scope:
  - Move storage toggle/volume-selection writes to storage gateway APIs.
  - Keep settings UI as intent-only layer (request change, render result).
  - Align with `PreferencesGateway` migration where required.
- Out of scope:
  - Non-storage settings refactors.
