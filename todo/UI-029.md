## Problem

Repeated visual literals (`zIndex`, shape radius, snackbar spacing/elevation) reduce consistency and increase friction for theme/layout maintenance.

## Scope

- In scope:
  - Extract repeated `GameGrubMain` overlay/snackbar literals to named UI tokens.
  - Keep one-off values inline when they are not reused.
  - Preserve visual behavior.
- Out of scope:
  - Full design-system overhaul.
