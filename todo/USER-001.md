## User Story

As a new user, when I launch the app for the first time, I want to see a clean, intuitive onboarding experience that guides me through getting started so I can quickly understand how to connect my game accounts and start playing.

## Problem

Currently, a new user opening the app sees an empty or confusing state with no clear path to get started. There's no onboarding flow to explain what the app does and how to connect their accounts.

## Scope

- In scope:
  - First-launch welcome screen with app overview
  - Quick setup wizard
  - Clear call-to-action to connect at least one store account
  - Brief explanation of what the app does
- Out of scope:
  - Deep customization options during onboarding
  - Tutorial videos

## Acceptance Criteria

- [ ] First-time users see a welcome/onboarding screen
- [ ] Onboarding explains basic app functionality
- [ ] Onboarding provides clear path to connect accounts
- [ ] Onboarding completes in under 60 seconds
- [ ] User can skip onboarding if desired

## Validation

- [ ] Manual flow: Fresh install on device shows onboarding
- [ ] User can complete onboarding and reach library
- [ ] User can skip onboarding and reach library
