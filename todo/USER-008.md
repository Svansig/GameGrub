## User Story

As a user, I want to search for games in my library so that I can quickly find a specific game without scrolling through everything.

## Problem

Users with large libraries need a quick way to find specific games.

## Scope

- In scope:
  - Search input field in library view
  - Real-time filtering as user types
  - Search by game title (case-insensitive)
  - Clear search button to reset

## Acceptance Criteria

- [ ] Search bar is accessible from library view
- [ ] Typing in search filters library in real-time
- [ ] Search matches partial titles
- [ ] Clear button resets to full library
- [ ] Search works across all store filters

## Validation

- [ ] Manual flow: Search for existing game by name
- [ ] Manual flow: Verify results are accurate
- [ ] Manual flow: Clear search and return to full library
