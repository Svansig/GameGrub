## User Story

As a user, I want to sort my game library so that I can organize how games are displayed (alphabetically, recently played, etc.).

## Problem

Users may want different ordering of their games.

## Scope

- In scope:
  - Sort options: Alphabetical (A-Z), Alphabetical (Z-A), Recently Played, Store
  - Sort option selector in library view
  - Sort preference persists across sessions

## Acceptance Criteria

- [ ] Sort options are accessible from library
- [ ] Selecting a sort option reorders the library
- [ ] Sort preference persists across app restarts
- [ ] Active sort option is indicated

## Validation

- [ ] Manual flow: Change sort order
- [ ] Manual flow: Verify order changes correctly
- [ ] Manual flow: Restart app and verify sort persists
