## User Story

As a user, I want to install a game from my library so that I can play it on my device.

## Problem

Users need to download and install games to play them.

## Scope

- In scope:
  - Install option on game detail screen
  - Download progress display (percentage, size downloaded/total)
  - Install location selection (internal vs external storage if available)
  - Cancel installation option
  - Resume interrupted downloads
  - "Installing" status indicator on game

## Acceptance Criteria

- [ ] User can initiate game installation from game details
- [ ] Download progress is visible and accurate
- [ ] User can cancel ongoing installation
- [ ] User can pause and resume download
- [ ] Game appears as "Installing" during download
- [ ] Completed installation shows "Installed" status
- [ ] Installation completes without errors for supported games

## Validation

- [ ] Manual flow: Initiate game installation
- [ ] Manual flow: Observe download progress
- [ ] Manual flow: Complete installation and verify game playable
