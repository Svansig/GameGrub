## User Story

As a user, I want to update an installed game to the latest version so that I can play with the newest content and fixes.

## Problem

Games need to be updated when new versions are available.

## Scope

- In scope:
  - Detection of available updates
  - Update option on game detail screen
  - Download progress for updates
  - Update notification/indicator
  - Auto-update option (settings)

## Acceptance Criteria

- [ ] Available updates are detected and indicated
- [ ] User can manually check for updates
- [ ] User can initiate game update
- [ ] Update download progress is visible
- [ ] User can enable auto-updates in settings
- [ ] Update completes successfully

## Validation

- [ ] Manual flow: Check for game updates
- [ ] Manual flow: Complete game update
- [ ] Manual flow: Enable auto-update and verify it works
