## User Story

As a user, I want to verify the integrity of my installed game files so that I can fix corrupted or missing files.

## Problem

Installed games may have corrupted or missing files that prevent them from working properly.

## Scope

- In scope:
  - "Verify files" option in game options
  - Progress indicator during verification
  - Repair/re-download of corrupted files
  - Results summary after verification

## Acceptance Criteria

- [ ] User can initiate file verification from game options
- [ ] Verification progress is displayed
- [ ] Corrupted files are automatically re-downloaded
- [ ] Summary shown after verification completes
- [ ] User can use this to troubleshoot launch issues

## Validation

- [ ] Manual flow: Initiate file verification
- [ ] Manual flow: Verify progress is shown
- [ ] Manual flow: Complete verification
