## User Story

As a user, I want to apply known compatibility fixes to games that don't work out of the box so that I can play more of my library.

## Problem

Some games require specific fixes or workarounds to function properly.

## Scope

- In scope:
  - Database of known game fixes
  - Auto-suggest fixes for known problematic games
  - Manual fix application
  - View applied fixes
  - Remove/revert fixes

## Acceptance Criteria

- [ ] Known game fixes are available in the app
- [ ] User can view available fixes for a game
- [ ] User can apply fixes to games
- [ ] User can remove applied fixes
- [ ] Fixes are applied automatically when launching if available

## Validation

- [ ] Manual flow: View available fixes for problematic game
- [ ] Manual flow: Apply fix and launch game
