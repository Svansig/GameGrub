## User Story

As a user, I want to see how much storage space my games are using so that I can manage my device storage.

## Problem

Users need to know which games are using how much space.

## Scope

- In scope:
  - Per-game storage usage display
  - Total storage used indicator
  - Available storage display
  - Sort by size option

## Acceptance Criteria

- [ ] User can see storage used by each game
- [ ] User can see total storage used
- [ ] User can see available storage
- [ ] User can sort games by size

## Validation

- [ ] Manual flow: View storage info in library
- [ ] Manual flow: Verify accuracy against actual files
