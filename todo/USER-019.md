## User Story

As a user, I want to move an installed game to different storage (internal to external or vice versa) so that I can manage my storage space.

## Problem

Users may need to move games between internal and external storage.

## Scope

- In scope:
  - Move game option in game options
  - Storage destination selection
  - Move progress indicator
  - Cancel move option

## Acceptance Criteria

- [ ] User can initiate move from game options
- [ ] User can select destination storage
- [ ] Progress is shown during move
- [ ] Game remains playable after move
- [ ] User can cancel move if needed

## Validation

- [ ] Manual flow: Move game to external storage
- [ ] Manual flow: Move game back to internal storage
