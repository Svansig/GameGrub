## User Story

As a user, I want to add a custom game (not from supported stores) to my library so that I can launch it using the app's container.

## Problem

Users may have games from other sources they want to manage in the app.

## Scope

- In scope:
  - Add custom game option
  - Specify executable path
  - Set display name
  - Configure launch options
  - Import artwork (manual or from SteamGridDB)

## Acceptance Criteria

- [ ] User can add custom game to library
- [ ] User can specify game name and executable
- [ ] User can configure launch parameters
- [ ] User can import artwork
- [ ] Custom games appear in library

## Validation

- [ ] Manual flow: Add custom game
- [ ] Manual flow: Launch custom game
