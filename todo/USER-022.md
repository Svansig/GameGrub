## User Story

As a user, I want to disconnect a store account from the app so that I can remove access or switch to a different account.

## Problem

Users may need to disconnect accounts for various reasons.

## Scope

- In scope:
  - Disconnect option in account settings
  - Confirmation dialog
  - Remove associated credentials
  - Option to keep or remove local data

## Acceptance Criteria

- [ ] User can disconnect account from settings
- [ ] Confirmation dialog prevents accidental disconnect
- [ ] Credentials are removed
- [ ] Games from that account are removed from library
- [ ] User can reconnect later with same or different account

## Validation

- [ ] Manual flow: Disconnect Steam account
- [ ] Manual flow: Reconnect with different account
