## User Story

As a user, I want to see a home screen that shows me the most important information at a glance so that I can quickly get started playing.

## Problem

Users need a landing page that provides quick access to key features.

## Scope

- In scope:
  - Recently played games section
  - Quick access to library
  - Connected accounts status
  - Active downloads status
  - Quick actions (continue playing, browse library)

## Acceptance Criteria

- [ ] Home screen displays after app launch
- [ ] Recently played games are shown
- [ ] User can quickly access library
- [ ] Connected accounts status is visible
- [ ] Active downloads are shown

## Validation

- [ ] Manual flow: Open app, view home screen
- [ ] Manual flow: Navigate to library from home
