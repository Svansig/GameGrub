## User Story

As a user, I want to interact with a running game using touch controls or a controller so that I can play the game.

## Problem

Users need input methods to control games.

## Scope

- In scope:
  - Touch screen controls (tap, swipe, pinch)
  - Controller input support
  - On-screen keyboard
  - Overlay menu (accessed via gestures)
  - Performance HUD toggle

## Acceptance Criteria

- [ ] Touch controls work in games
- [ ] Controller is recognized and works
- [ ] On-screen keyboard available when needed
- [ ] Overlay menu accessible during gameplay
- [ ] Performance HUD can be toggled

## Validation

- [ ] Manual flow: Play game with touch
- [ ] Manual flow: Play game with controller
- [ ] Manual flow: Access overlay during play
