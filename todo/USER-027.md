## User Story

As a user, I want to manage DLC (downloadable content) for my games so that I can install or uninstall additional content.

## Problem

Users may want to add or remove DLC.

## Scope

- In scope:
  - View DLC list for game
  - Install/uninstall DLC
  - DLC download progress
  - DLC storage size

## Acceptance Criteria

- [ ] User can view available DLC for a game
- [ ] User can install DLC
- [ ] User can uninstall DLC
- [ ] DLC installation progress is shown

## Validation

- [ ] Manual flow: View DLC for game
- [ ] Manual flow: Install DLC
