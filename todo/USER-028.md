## User Story

As a user, I want to open a game's store page in an external browser so that I can view more information or make purchases.

## Problem

Users may want to access the store page for a game.

## Scope

- In scope:
  - "View in Store" option in game options
  - Opens in external browser
  - Links to correct store page

## Acceptance Criteria

- [ ] User can open store page from game options
- [ ] Correct store page opens in browser

## Validation

- [ ] Manual flow: Open store page from game
