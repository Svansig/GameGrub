## User Story

As a user, I want to create a shortcut on my Android home screen to a specific game so that I can launch it directly without opening the app.

## Problem

Users want quick access to frequently played games.

## Scope

- In scope:
  - Create shortcut option in game options
  - Creates Android home screen shortcut
  - Shortcut includes game artwork
  - Tapping shortcut launches game directly

## Acceptance Criteria

- [ ] User can create shortcut from game options
- [ ] Shortcut appears on Android home screen
- [ ] Shortcut displays game artwork
- [ ] Tapping shortcut launches game

## Validation

- [ ] Manual flow: Create shortcut
- [ ] Manual flow: Tap shortcut to launch game
