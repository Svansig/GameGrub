## User Story

As a user, I want to see all active downloads in one place so that I can monitor progress and manage them.

## Problem

Users may have multiple games downloading and need to track them.

## Scope

- In scope:
  - Downloads section/tab showing all active downloads
  - Individual progress per game
  - Total progress/aggregate view
  - Pause/resume all option
  - Cancel option

## Acceptance Criteria

- [ ] User can view all active downloads
- [ ] Each download shows individual progress
- [ ] User can pause/resume individual downloads
- [ ] User can cancel downloads
- [ ] User can pause/resume all

## Validation

- [ ] Manual flow: Start multiple downloads
- [ ] Manual flow: View downloads list
