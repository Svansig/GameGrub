## User Story

As a user, I want to configure the container resources (CPU cores, RAM) so that I can optimize performance for my device.

## Problem

Different devices need different resource allocations.

## Scope

- In scope:
  - CPU core allocation setting
  - RAM allocation setting
  - Per-game override option
  - Global default setting

## Acceptance Criteria

- [ ] User can set global CPU/RAM allocation
- [ ] User can override settings per-game
- [ ] Settings persist across sessions

## Validation

- [ ] Manual flow: Modify container resources
- [ ] Manual flow: Launch game with new settings
