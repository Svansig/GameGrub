## User Story

As a user, I want to select which Wine or Proton version to use so that I can maximize compatibility for my games.

## Problem

Different games work better with different Wine/Proton versions.

## Scope

- In scope:
  - List of available Wine/Proton versions
  - Version selection in settings
  - Per-game version override
  - Auto-select suggested version option

## Acceptance Criteria

- [ ] User can view available Wine/Proton versions
- [ ] User can select default version
- [ ] User can override per-game
- [ ] User can enable auto-select

## Validation

- [ ] Manual flow: Change Wine version
- [ ] Manual flow: Launch game with different version
