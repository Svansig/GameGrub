## User Story

As a user, I want to configure display settings so that games look better on my device.

## Problem

Display settings affect game visuals and performance.

## Scope

- In scope:
  - Resolution settings
  - Fullscreen toggle
  - Orientation (landscape/portrait/auto)
  - DPI scaling
  - Graphics API selection (Vulkan/OpenGL)

## Acceptance Criteria

- [ ] User can configure resolution
- [ ] User can toggle fullscreen
- [ ] User can set orientation preference
- [ ] User can select graphics API

## Validation
