## User Story

As a user, I want to configure the touch input overlay so that I can play games that don't support controllers.

## Problem

Some games require touch input and need an on-screen control overlay.

## Scope

- In scope:
  - Enable/disable overlay
  - Overlay position/size adjustment
  - Button layout customization
  - Opacity setting
  - Save/load overlay profiles

## Acceptance Criteria

- [ ] User can enable/disable input overlay
