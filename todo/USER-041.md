## User Story

As a user, I want to quickly switch between running games or minimize a game to do something else so that I can multitask.

## Problem

Users may need to switch between games or access the app while a game is running.

## Scope

- In scope:
  - Minimize game to background
  - Return to running game
  - App stays running in background
  - Notification showing game is running

## Acceptance Criteria

- [ ] User can minimize running game
- [ ] User can return to running game
- [ ] Game state is preserved
- [ ] App can be accessed while game runs

## Validation

- [ ] Manual flow: Launch game, minimize, return
