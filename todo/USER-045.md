## User Story

As a user, I want to submit feedback about a game's compatibility so that the community can benefit from my experience.

## Problem

Users want to share their experiences to help others.

## Scope

- In scope:
  - Feedback option in game options
  - Rating (works/doesn't work/partially works)
  - Notes field
  - Optional: include configuration
  - Anonymous submission option

## Acceptance Criteria

- [ ] User can submit feedback for a game
- [ ] User can rate compatibility
- [ ] User can add notes
- [ ] Submission is anonymous by default
