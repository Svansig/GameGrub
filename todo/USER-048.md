## User Story

As a user, I want to import and export game configurations so that I can share settings with others or backup my configurations.

## Problem

Users want to share their working configurations or back them up.

## Scope

- In scope:
  - Export single game config to file
  - Export all game configs to file
  - Import config from file
  - Config includes launch settings, fixes, etc.

## Acceptance Criteria

- [ ] User can export game config to file
- [ ] User can import game config from file
- [ ] Imported config applies correctly

## Validation

- [ ] Manual flow: Export config
- [ ] Manual flow: Import config to different game
