## User Story

As a user, I want to clear the app cache so that I can free up storage space and troubleshoot issues.

## Problem

Cached data can accumulate and take up space.

## Scope

- In scope:
  - Clear image cache option
  - Clear download cache option
  - Clear temp files option
  - Show cache size before clearing

## Acceptance Criteria

- [ ] User can view cache sizes
- [ ] User can clear various caches
- [ ] Clearing frees storage space
- [ ] App functions normally after cache clear

## Validation

- [ ] Manual flow: Clear cache
- [ ] Manual flow: Verify space freed
