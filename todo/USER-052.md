## User Story

As a user, I want notifications to be handled appropriately during gameplay so that I'm not interrupted but don't miss important alerts.

## Problem

Notifications can interfere with games or be missed during play.

## Scope

- In scope:
  - Mute notifications during gameplay option
  - Show important notifications (downloads, etc.)
  - Notification sound/vibration settings

## Acceptance Criteria

- [ ] User can enable/disable game mode
- [ ] Critical notifications still appear
- [ ] User can configure notification behavior

## Validation

- [ ] Manual flow: Test notifications during game
