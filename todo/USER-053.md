## User Story

As a user, I want to see a visual representation of controller buttons while playing so that I know which buttons to press.

## Problem

Users may forget button mappings during gameplay.

## Scope

- In scope:
  - Controller layout overlay option
  - Toggle overlay on/off during play
  - Customize which buttons are shown

## Acceptance Criteria

- [ ] User can enable controller layout overlay
- [ ] Overlay shows button mappings
- [ ] User can toggle overlay during play

## Validation

- [ ] Manual flow: Enable overlay and verify display
