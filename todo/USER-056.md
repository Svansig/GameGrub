## User Story

As a user, I want to view play statistics so that I can see how much time I've spent playing games.

## Problem

Players may want to track their gaming time.

## Scope

- In scope:
  - Total play time across all games
  - Play time per game
  - Last played date
  - Most played games list

## Acceptance Criteria

- [ ] User can view total play time
- [ ] User can view per-game play time
- [ ] Statistics are accurate
