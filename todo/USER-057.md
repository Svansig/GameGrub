## User Story

As a user, I want to see performance metrics while playing so that I can monitor FPS, CPU, and memory usage.

## Problem

Users may want to monitor performance during gameplay.

## Scope

- In scope:
  - FPS counter
  - CPU usage display
  - Memory usage display
  - Toggle HUD on/off
  - HUD position adjustment
  - Configure what metrics are shown

## Acceptance Criteria

- [ ] User can enable performance HUD
- [ ] HUD displays FPS, CPU, memory
- [ ] User can customize what metrics shown
- [ ] HUD doesn't significantly impact performance

## Validation

- [ ] Manual flow: Enable HUD during gameplay
