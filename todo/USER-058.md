## User Story

As a user, I want to configure the in-game overlay menu so that I can customize what options are available during gameplay.

## Problem

The overlay menu is the main way to interact with the app while playing, and users may want to customize it.

## Scope

- In scope:
  - Enable/disable specific menu items
  - Customize gesture to open overlay
  - Configure overlay appearance
  - Reset to defaults option

## Acceptance Criteria

- [ ] User can configure overlay menu items
- [ ] User can customize overlay trigger gesture
- [ ] Customizations persist
- [ ] Reset to defaults works
