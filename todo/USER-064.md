## User Story

As a user, when I try to launch a game with anti-cheat, I want to be informed about compatibility limitations so that I know if the game will work.

## Problem

Some games with anti-cheat systems may not work on the app.

## Scope

- In scope:
  - Detect anti-cheat in game
  - Show warning/information about compatibility
  - List known anti-cheat systems and their status
  - Links to community compatibility info

## Acceptance Criteria

- [ ] Anti-cheat games show compatibility warning
- [ ] User understands limitations
- [ ] Community compatibility info is accessible

## Validation

- [ ] Manual flow: Launch game with anti-cheat
