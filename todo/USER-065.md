## User Story

As a user, I want to reset a game's configuration to default so that I can start fresh if I've made too many customizations.

## Problem

Users may mess up game settings and want to reset.

## Scope

- In scope:
  - Reset to default option in game settings
  - Confirmation dialog
  - Reset launch settings, fixes, and configurations

## Acceptance Criteria

- [ ] User can reset game config to default
- [ ] Confirmation prevents accidental reset
- [ ] Game launches with default settings after reset

## Validation

- [ ] Manual flow: Reset game config
