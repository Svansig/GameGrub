## User Story

As a user, I want to backup and restore my app data so that I can preserve my settings and configurations when switching devices or reinstalling.

## Problem

Users need to preserve their data across reinstalls or device changes.

## Scope

- In scope:
  - Export all app data to file
  - Import app data from file
  - Include settings, configurations, accounts (optionally)
  - Selective backup (settings only, games only, etc.)

## Acceptance Criteria

- [ ] User can export all app data
- [ ] User can import data from backup file
- [ ] Backup includes settings and configurations
- [ ] User can selectively backup

## Validation

- [ ] Manual flow: Create backup
- [ ] Manual flow: Restore from backup
