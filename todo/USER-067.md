## User Story

As a Steam user, I want to browse and download Steam Workshop content so that I can add mods and user-generated content to my games.

## Problem

Steam Workshop content enhances games but may not be accessible.

## Scope

- In scope:
  - Browse workshop for installed games
  - Search workshop content
  - Download and install workshop items
  - Manage installed workshop content

## Acceptance Criteria

- [ ] User can browse workshop for a game
