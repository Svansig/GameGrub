## User Story

As a user, I want to quickly see only my installed games so that I can find games to play without scrolling through my entire library.

## Problem

Users with large libraries may want to filter to installed games only.

## Scope

- In scope:
  - "Installed" filter in library
  - Show only installed games
  - Clear indication of filter state

## Acceptance Criteria

- [ ] User can filter to installed games only
- [ ] Filter state is clearly indicated
- [ ] User can return to all games view

## Validation

- [ ] Manual flow: Filter to installed games
