## User Story

As a user, I want to see community ratings for game compatibility so that I know which games work well before installing.

## Problem

Users want to know which games are known to work before spending time downloading.

## Scope

- In scope:
  - Display community compatibility rating
  - Show number of users who reported
  - Show common issues if any
  - Allow user to rate after playing

## Acceptance Criteria

- [ ] Compatibility rating shown on game
- [ ] User can see how many reports
- [ ] User can see common issues
- [ ] User can submit their rating

## Validation

- [ ] Manual flow: View compatibility rating
