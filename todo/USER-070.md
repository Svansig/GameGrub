## User Story

As a user, I want to manage my save files for a game so that I can backup, restore, or delete saves.

## Problem

Users may want to manage their saves manually.

## Scope

- In scope:
  - View list of save files
  - Backup saves to local file
  - Restore saves from backup file
  - Delete individual saves
  - View save file size

## Acceptance Criteria

- [ ] User can view all save files for a game
- [ ] User can backup saves to file
- [ ] User can restore saves from file
- [ ] User can delete saves

## Validation

- [ ] Manual flow: Backup saves
- [ ] Manual flow: Restore saves
