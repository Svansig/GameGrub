## User Story

As a user, I want to pin my favorite games to the top of my library so that I can find them quickly.

## Problem

Users have favorite games they play frequently.

## Scope

- In scope:
  - Pin game to top option
  - Multiple games can be pinned
  - Unpin game option
  - Pinned games persist

## Acceptance Criteria

- [ ] User can pin games to top
- [ ] Pinned games appear at top of library
- [ ] User can unpin games
- [ ] Pin state persists

- [ ] Manual flow: Pin game
- [ ] Manual flow: Unpin game
