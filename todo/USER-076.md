## User Story

As a user with multiple accounts on the same store, I want to switch between them so that I can access different game libraries.

## Problem

Users may have multiple Steam, GOG, Epic, or Amazon accounts.

## Scope

- In scope:
  - Switch between accounts for same store
  - Show which account is currently active
  - Quick switch option in library

## Acceptance Criteria

- [ ] User can switch to different account
- [ ] Library updates to show new account's games
- [ ] User can see which account is active

## Validation

- [ ] Manual flow: Add second account, switch between accounts
