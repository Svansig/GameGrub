## User Story

As a user, I want to use mouse and keyboard with my Android device so that I can play games that require these inputs.

## Problem

Some PC games need keyboard and mouse, not just touch or controller.

## Scope

- In scope:
  - USB OTG keyboard/mouse support
  - On-screen keyboard for games
  - Mouse pointer emulation
  - Key mapping configuration

## Acceptance Criteria

- [ ] External keyboard works in games
- [ ] External mouse works in games
- [ ] On-screen keyboard available
- [ ] User can map keys

## Validation

- [ ] Manual flow: Use USB keyboard in game
