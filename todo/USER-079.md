## User Story

As a user, I want to configure battery optimization settings so that the app continues to work reliably in the background.

## Problem

Android battery optimization can kill background services needed for downloads and gaming.

## Scope

- In scope:
  - Battery optimization status indicator
  - Link to disable battery optimization
  - Explanation of why it's needed

## Acceptance Criteria

- [ ] User can see battery optimization status
- [ ] User can easily access settings to disable
- [ ] Background downloads work with optimization disabled
