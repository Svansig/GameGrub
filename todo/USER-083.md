## User Story

As a user, I want to create and save custom controller profiles so that I can quickly switch between different control schemes.

## Problem

Different games or use cases may need different controller configurations.

## Scope

- In scope:
  - Create new controller profile
  - Save button mappings to profile
  - Name and describe profile
  - Load profile when playing
  - Delete profile

## Acceptance Criteria

- [ ] User can create new profile
- [ ] User can save current bindings to profile
- [ ] User can load profile during gameplay
- [ ] User can delete profiles

## Validation

- [ ] Manual flow: Create and load controller profile
