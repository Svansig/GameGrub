## User Story

As a new user, I want to connect my game accounts during onboarding with minimal friction so that I can start playing immediately without additional setup steps.

## Problem

The current account connection flow requires too many steps and doesn't guide users effectively through the process.

## Scope

- In scope:
  - Streamlined OAuth flows that complete in fewer taps
  - Clear progress indicator during connection
  - Auto-detect existing credentials
  - "Connect later" option without blocking onboarding
  - Immediate library preview after connection

## Acceptance Criteria

- [ ] Account connection completes in under 30 seconds
- [ ] User can connect account in 3 taps or fewer
- [ ] Progress is clearly shown during connection
- [ ] User can skip without feeling pressured

## Validation

- [ ] Manual flow: Complete account connection in onboarding
