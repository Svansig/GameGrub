## User Story

As a user, I want to install games with fewer taps and better progress visibility so that the process feels seamless and I know what's happening.

## Problem

The current install flow requires multiple confirmation steps and doesn't provide clear feedback.

## Scope

- In scope:
  - One-tap install from game card
  - Unified install progress across all stores
  - Background installation with notification updates
  - Estimated time remaining
  - Smart auto-start game when install completes

## Acceptance Criteria

- [ ] User can install with single tap
- [ ] Progress updates every 5 seconds
- [ ] User can start playing immediately after install
- [ ] Install continues even if app is closed

## Validation

- [ ] Manual flow: Install game with minimal taps
