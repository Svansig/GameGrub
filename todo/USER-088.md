## User Story

As a user with a large library, I want faster browsing with smart suggestions so that I can quickly find what to play.

## Problem

Large libraries load slowly and users spend too much time scrolling to find games.

## Scope

- In scope:
  - Instant search with result highlighting
  - "Recently updated" and "Recently played" smart sections
  - Quick filter chips (Recently Played, Installed, Favorites)
  - Smooth infinite scroll with lazy loading
  - Visual distinction for playable vs not-yet-installed

## Acceptance Criteria

- [ ] Library loads in under 2 seconds
- [ ] Search results appear as user types
- [ ] Smart sections appear on home screen
- [ ] Scrolling remains smooth with 500+ games

## Validation

- [ ] Manual flow: Browse library with 100+ games
