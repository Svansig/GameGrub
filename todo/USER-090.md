## User Story

As a user, I want game configuration to be simpler with smart defaults so that I only need to configure settings when something doesn't work.

## Problem

Users are forced to configure many settings before playing, and there's no guidance on what to change.

## Scope

- In scope:
  - "Just Works" default configuration for all games
  - Configuration wizard for games that need it
  - One-click "Apply recommended fix" for common issues
  - Easy reset to defaults
  - Import community configurations with one tap

## Acceptance Criteria

- [ ] 80%+ of games work with zero configuration
- [ ] Common fixes available in one click
- [ ] Configuration wizard guides users through needed changes
- [ ] User can reset all configs with one tap

## Validation

- [ ] Manual flow: Launch game with default config
