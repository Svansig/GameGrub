## User Story

As a user, I want a consistent authentication experience across all stores so that connecting accounts feels the same regardless of which store I'm adding.

## Problem

Each store has different authentication flows with varying quality and UX.

## Scope

- In scope:
  - Unified connection UI pattern for all stores
  - Consistent error handling across platforms
  - Token refresh handled automatically in background
  - Clear status indicators for each connected account
  - Easy account switching within same store

## Acceptance Criteria

- [ ] All stores use same connection flow pattern
- [ ] Errors are consistent and actionable
- [ ] User can see all account states at a glance
- [ ] Session refresh happens transparently

## Validation

- [ ] Manual flow: Connect all 4 stores
