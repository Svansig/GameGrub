## User Story

As a user, I want quick access to common actions on installed games so that I don't have to navigate through menus.

## Problem

Users need multiple taps to access common actions like launch, update, or verify.

## Scope

- In scope:
  - Long-press quick action menu on game cards
  - Swipe actions (launch on right, options on left)
  - "Play" button prominent on installed games
  - Batch actions for multiple selected games

## Acceptance Criteria

- [ ] User can launch game with single tap
- [ ] Long-press reveals quick actions
- [ ] Swipe gestures work for common actions
- [ ] Multiple games can be selected for batch operations

## Validation

- [ ] Manual flow: Use quick actions on installed game
