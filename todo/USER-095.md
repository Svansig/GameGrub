## User Story

As a user with multiple games downloading, I want intelligent queue management so that I can prioritize and control downloads easily.

## Problem

Downloads are linear and don't adapt to user needs or network conditions.

## Scope

- In scope:
  - Drag-and-drop queue reordering
  - Pause/resume individual or all downloads
  - Auto-pause when user starts playing a game
  - Smart prioritization based on recency of interaction
  - Download on Wi-Fi only option with mobile override

## Acceptance Criteria

- [ ] User can reorder downloads via drag-drop
- [ ] Active game automatically pauses other downloads
- [ ] Downloads pause on cellular by default
- [ ] User can override Wi-Fi-only for individual downloads

## Validation

- [ ] Manual flow: Manage download queue
