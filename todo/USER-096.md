## User Story

As a user, I want to discover games across all my connected stores in one place so that I don't miss games I might want to play.

## Problem

Users have to check each store separately to see what's available.

## Scope

- In scope:
  - Unified "Browse All" across stores
  - "Not in library" suggestions
  -跨 Store search
  - Recommendations based on owned games

- [ ] User can see recommendations
