## User Story

As a user, I want to quickly resume my last game session so that I can get back to playing with minimal wait time.

## Problem

Starting a game takes too long, especially for games the user plays frequently.

## Scope

- In scope:
  - "Continue Playing" on home screen
  - Instant resume for recently played games
  - Remember exact position in game
  - Quick launch from notification

## Acceptance Criteria

- [ ] Last played game shown on home screen
- [ ] User can resume in under 5 seconds
- [ ] Game state is preserved exactly

## Validation

- [ ] Manual flow: Resume last game
